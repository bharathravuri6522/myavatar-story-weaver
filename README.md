## ✨ Features

### Core Features

- **Story Setup**: Title, Genre selection, Initial Hook
- **AI Opening Paragraph**: Strong 150–250 word hook generated by AI
- **Collaborative Storytelling**: Continue with AI or add your own contributions
- **Give Me Choices**: AI suggests 3 branching story directions
- **Full Context Memory**: Entire story history sent with every call for maximum consistency
- **Temperature Slider**: Control creativity level (Low = Consistent, High = Creative)

### Bonus Features Implemented

- **Live Character Tracker**: Automatically extracts and displays all characters with descriptions and arcs
- **Genre Remix**: Change genre mid-story while preserving characters and plot
- **Undo Last AI Turn**
- **Visualization Prompt Generator** (ready for Flux / Midjourney / DALL·E)
- **Export Story as Markdown**
- **Strong Guardrails** to keep the AI in storyteller mode
- **Rate Limit Handling**

---

## Tech Stack

- **Framework:** Streamlit
- **LLM:** Groq + Llama 3.1 8B Instant (fast & high daily limits)
- **Memory Strategy:** Full story history sent with every API call (as per challenge requirement)
- **Structured Output:** JSON mode for reliable character extraction

## 🚀 Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/myavatar-story-weaver.git
cd myavatar-story-weaver
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Add Groq API Key

**Change the .env.example file in the root folder to .env and add your API Key:**

```bash
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 5. Run the app

```bash
streamlit run app.py
```
